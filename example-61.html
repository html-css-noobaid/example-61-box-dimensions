CSS Box Tutorial

Example #1: Box Dimensions

CSS has properties that allows you to set the height and width of boxes, as well as the minimum/maximum height and width. We use the overflow property to decide whether we want a scrollbar or we want to cut off text that does not fit in the box.

The display property allows us to decide if we want our box to act like a block element where each box starts on a new line or an inline element where we can have several boxes on the same line.

The margin properties (margin-top, margin-right, margin-bottom, margin-left) allow us to control the gaps between boxes.
